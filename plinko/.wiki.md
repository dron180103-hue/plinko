# Project Summary
**Plinko Idle Clicker** is an engaging web-based game that combines chance and strategy, allowing players to drop balls onto a dynamic board to earn rewards. The game incorporates user account management, competitive leaderboards, and in-game purchases, supported by a robust backend for user authentication and data management. The project aims to enhance player engagement through competitive gameplay and a visually appealing interface, while shifting from real currency to in-game tokens to avoid casino-like associations.

# Project Module Description
- **Backend**: Manages user authentication, game state persistence, leaderboard management, and health checks.
- **Frontend**: Offers an intuitive interface for gameplay, account management, and visual feedback, optimized for mobile devices.

# Directory Tree
```
app/
├── backend/
│   ├── core/                     # Core application logic and configurations

    ├── src/                      # Source code for the frontend application
```

# File Description Inventory
- **app/backend/main.py**: Main entry point for the backend server.
- **app/backend/routers/**: Contains API routes for user authentication and game actions, including `game.py` for player-related operations.
- **app/frontend/src/**: Contains React components, utilities, and styles for the frontend application.
- **app/frontend/public/**: Static assets such as images and icons for the frontend.
- **app/backend/requirements.txt**: Lists Python dependencies for the backend application.
- **app/frontend/src/lib/api.ts**: Manages API interactions, including error handling and connection status tracking.
- **app/frontend/src/pages/Index.tsx**: Main game interface managing game logic, state, and user interactions.
- **app/frontend/public/sw.js**: Service worker script for caching assets and managing offline functionality.
- **app/frontend/src/lib/i18n.ts**: Manages localization and referral URL configuration for ad buttons.
- **app/frontend/src/components/GameTab.tsx**: Component for the game tab interface.
- **app/frontend/src/components/PlinkoCanvas.tsx**: Component for rendering the Plinko game board.

# Technology Stack
- **Backend**: FastAPI, SQLAlchemy, PostgreSQL, Alembic for database migrations.
- **Frontend**: React, TypeScript, Tailwind CSS, Vite for modern frontend tooling.

# Usage
1. **Install Dependencies**:
   - For backend: Navigate to `app/backend` and install Python dependencies listed in `requirements.txt`.
   - For frontend: Navigate to `app/frontend` and install JavaScript dependencies.

2. **Build the Project**:
   - In the frontend directory, compile the frontend assets.

3. **Run the Application**:
   - Start the backend server by executing the command in `app/backend/main.py`.
   - For the frontend, serve the built assets.
