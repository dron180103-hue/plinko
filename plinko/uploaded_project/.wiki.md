# Project Summary
The project is developing a **Multi-Language Gambling Ecosystem** that integrates a Telegram Bot and a Mini App for an automated gambling referral system. This ecosystem aims to enhance user engagement through an interactive Plinko-style game within the Mini App, allowing users to earn rewards. The system supports multiple languages (RU, UA, EN) and utilizes a Python backend with SQLite for user data persistence. Recent updates have transformed the gameplay into a more comprehensive product, including a shop system for purchasing skins, themes, and boosters, alongside enhanced user retention mechanics.

# Project Module Description
The project consists of two main components:
1. **Telegram Bot**: Manages user interactions, onboarding, and provides access to the Mini App.
2. **Mini App**: A Plinko-style game where users can earn "Demo Coins" and engage in gameplay. Key features include:
   - Multi-ball drop system with selectable ball counts.
   - In-game shop for purchasing skins, themes, boosters, and special items.
   - Enhanced gameplay mechanics, including daily and weekly missions, a fortune wheel, and a prestige system.
   - User interface improvements and mobile optimization.

# File Description Inventory
- **README.md**: Project overview and instructions.
- **index.html**: Main HTML file for the application.
- **index.css**: Styles for the application.
- **package.json**: Project dependencies and scripts.
- **postcss.config.js**: Configuration for PostCSS.
- **src/**: Contains the main source code, including pages and application logic.
- **seo-scripts/**: Scripts for SEO-related tasks.
- **public/**: Public assets like favicon and robots.txt.
- **components/**: Contains UI components like game controls, conversion popups, and settings panel.
- **lib/**: Contains utility functions for game state management, internationalization, sound effects, and Telegram integration.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **State Management**: LocalStorage for game state persistence
- **Background Processing**: Web Workers for auto-production calculations
- **Backend**: Python (Aiogram 3.x) with SQLite for user data

# Usage
To set up and run the project:
1. Install dependencies: `pnpm install`
2. Build the project: `pnpm run build`
